# Threadbase

A personal digital closet and laundry tracker — log clothing items across
closets, track wear/wash status, and get help sorting laundry into
optimally grouped wash loads.

Built as a full-stack practice project, with a deliberate focus on 
AI-assisted development using Claude Code.

## Status

In early development — currently setting up core CRUD (closets and items).

## Planned Features

- Log clothing items by closet, type, brand, and colour
- Track wear/wash status (clean → light → heavy → dirty) per item, with
  suggested loads generated automatically based on care compatibility
  (temperature, colour, delicates)
- Photo uploads per item
- Care-tag photo scanning via Claude's vision API to auto-fill wash
  instructions
- Installable as a PWA for daily use on both desktop and phone

## Tech Stack

- **Frontend:** React (Vite)
- **Backend:** Node.js / Express
- **Database:** MongoDB Atlas
- **Image storage:** Cloudflare R2
- **Vision/OCR:** Claude API
